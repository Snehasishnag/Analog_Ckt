# BJT Current Mirror

A BJT-based current mirror circuit designed to replicate a reference current at the output using matched bipolar junction transistors (BJTs). Current mirrors are widely used in analog integrated circuits for biasing, active loads, and current sources due to their ability to provide stable and consistent current.

## Overview

This project demonstrates the design and analysis of a basic BJT current mirror. The circuit mirrors a reference current from one transistor to another while maintaining nearly constant output current under varying load conditions.

## Features

- Basic BJT Current Mirror
- Stable current replication
- Matched transistor configuration
- Suitable for analog biasing applications
- Circuit analysis and simulation results

## Working Principle

The circuit uses two matched BJTs with their bases and emitters connected together. One transistor is configured as a diode-connected transistor to establish the reference current. The second transistor mirrors this current, producing an output current approximately equal to the reference current.
