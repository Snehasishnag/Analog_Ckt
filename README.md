# Analog_Ckt

A curated collection of analog electronic circuit projects developed to explore fundamental and advanced concepts in analog circuit design. This repository serves as a growing portfolio featuring circuit analysis, simulation, and documentation for a wide range of analog applications.

---

## 📂 Repository Contents

| Project | Description | Status |
|---------|-------------|:------:|
| **3-Band Audio Mixer** | Three-channel audio mixer with gain analysis and complete project documentation. | ✅ Completed |
| **BJT Current Mirror** | Basic bipolar junction transistor (BJT) current mirror demonstrating accurate current replication for analog biasing applications. | ✅ Completed |
